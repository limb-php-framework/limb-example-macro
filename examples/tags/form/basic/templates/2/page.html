Form Tag Example

Description:

This page demonstrates how to use {{form:errors}} < and {{form:field_errors}} < tags.

Try to send an empty form and you will see errors list above the form. Note the title attribute of {{input}} tag - it used in error messages to display correct input field name text)) { ?>

You typed: {$#text} {{form name='my_form' method='POST'}} {{form:errors to='$errors'}} {{list using='$errors'}} Form errors:

    {{list:item}}
  • {$item.message}
    • {{/list:item}}
{{/list}} {{form:field_errors to='$my_field_errors'}} {{list using="$my_field_errors"}} Field errors (you can place field error near form field):

    {{list:item}}
  • (for field "{$item.id}"){$item.message}
    • {{/list:item}}
{{/list}} {{label for='my_input' error_class='error'}}Type here:
{{/label}} {{input type='text' id='my_input' name='my_input' title='Field Full Name' error_class='error_input'/}} {{/form}}

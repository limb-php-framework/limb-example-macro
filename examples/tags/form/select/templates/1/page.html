{{select}} tag example

Description:

This page demonstrates how to use {{select}} tag. We will show how to fill options list in different ways.

Options are listed in the template

{{form name='my_form1' method='POST' from='$#my_form1'}} my_form1['my_select1'])) { ?>

You selected: {$#my_form1.my_select1}

Select an option: {{select name='my_select1' type='text'}} {{option value='no_value' selected='true' }}------{{/option}} {{option value='first'}}First option{{/option}} {{option value='second'}}Second option{{/option}} {{/select}} {{/form}}

Options are passed into runtime select helper from php-script

{{form name='my_form2' method='POST' from='$#my_form2'}} my_form2['my_select2'])) { ?>

You selected: {$#my_form2.my_select2}

Select an option: {{select name='my_select2' type='text' options='$#options'/}} {{/form}}

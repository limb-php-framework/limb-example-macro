List Properties Example

A simple list formatted into a table.

Here we used some of the {{list}} tag properties like counter and parity

{{list using='$#php_modules' parity='$Parity' counter='$row_number'}} {{list:item}} {{/list:item}}
{$row_number} {$item.name} {$item.description|default:" "}
{{/list}}

The second example shows how to use {{list:odd}} and {{list:even}} tags

{{list using='$#php_modules'}} {{list:item}} {{list:odd}}{{/list:odd}} {{list:even}}{{/list:even}} {{/list:item}}
{$item.name} {$item.description|default:" "}
{{/list}}

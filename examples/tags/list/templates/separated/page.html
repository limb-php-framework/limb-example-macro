List Separator Example

This example show how to use {{list:glue}} and {{list:fill}}tags

Simple separator

{{list using='$#php_modules'}} {{list:item}} {$item.name} {{list:glue}}
{{/list:glue}} {{/list:item}} {{/list}}

Separator with defined step

{{list using='$#php_modules'}}
    {{list:item}}
  • {$item.name}
    • {{list:glue every='2'}}
    {{/list:glue}} {{/list:item}}
{{/list}}

Valid layout with multi column output

{{list using='$#php_modules'}} {{list:item}} {{list:glue every='4' literal='true'}}{{/list:glue}} {{/list:item}} {{list:fill upto='4' items_left='$count'}} {{repeat times="{$count}"}}{{/repeat}} {{/list:fill}}
{$item.name}
 
{{/list}}

Several separators

{{list using='$#php_modules'}}
{{list:item}} {$item.name} {{list:glue every='3'}}{{/list:glue}} {{list:glue}}
{{/list:glue}} {{/list:item}}
{{/list}}
